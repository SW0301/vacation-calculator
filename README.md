# Vacation Calculator API

**Vacation Calculator** — это микросервис на основе Spring Boot, предназначенный для расчета отпускных сотрудников. Он может рассчитывать отпускные на основе средней зарплаты и либо количества дней отпуска, либо точных дат начала и окончания отпуска (с учетом праздников и выходных).

## Функциональность

- **Основной функционал**: расчет отпускных на основе средней зарплаты и количества дней отпуска.
- **Дополнительный функционал**: расчет с учетом выходных и праздничных дней при указании точных дат отпуска.

## Технологии

- Java 11
- Spring Boot
- Maven

## URL приложения

Приложение будет доступно по адресу:  `http://localhost:8080`.

## API Описание

### Расчет отпускных

**Метод**: `GET /calculate`

#### Параметры:

- `averageSalary` (обязательный): Средняя зарплата за последние 12 месяцев.
- `vacationDays` (опциональный): Количество дней отпуска.
- `startDate` и `endDate` (опциональные): Начальная и конечная даты отпуска в формате `yyyy-MM-dd`. Оба поля должны быть указаны.

#### Примеры запросов:

1. **Расчет по количеству дней отпуска:**:
    ```bash
    GET /calculate?averageSalary=50000&vacationDays=10
    ```

2. **Расчет по датам начала и окончания отпуска:**:
    ```bash
    GET /calculate?averageSalary=50000&startDate=2024-05-01&endDate=2024-05-15
    ```
#### Ошибки
1. **Если передана только одна дата(`startDate` или `endDate`)**:
   ```
    Необходимо указать либо дни отпуска, либо как начальную, так и конечную дату.
    ```
2. **Если `startDate` позже чем `endDate`**:
   ```
    Дата начала отпуска не может быть позже даты окончания.
    ```

3. **Если не переданы ни `vacationDays`, ни даты**:ъ
   ```
    Как начальная, так и конечная дата должны указываться вместе.
    ```
